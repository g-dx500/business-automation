# sql

業務自動化に関するSQLサンプルコードです。

## sales_summary
事業者IDごとに、以下の売上集計を行うクエリです。

- 直近12か月の合計売上高（前年・今年それぞれ）
- 年初来の合計売上高（前年・今年それぞれ）
- 直近30日の合計売上高
- 直近7日の合計売上高

## sales_promotion_summary_report
事業者IDごとに、販促タイプ別の適用商品数を集計するクエリです。
直近7日間のデータを対象に、条件を満たす商品をタイプ別にカウントします。

## weekend_processing_report
事業者IDごとに、週末（土曜・日曜）の商品取扱割合を集計するクエリです。
直近4週間のデータを対象に、特定企画注文を除外したうえで集計します。
