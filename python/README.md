# python

業務自動化に関するPythonサンプルコードです。

## scraping_email_addresses
事業者IDのリストを基に、対象サイト上で各事業者の個別ページを
自動的に開き、メールアドレスを取得します。
複数の事業者IDに対してこの処理を自動で繰り返します。

使用技術：Selenium

## create_upload_data
2つのダウンロードデータを基に、DuckDBで加工・判定を行い、
システムへのアップロード用Excelファイルを自動作成します。

- 条件フィルタリングおよびティア分類
- テストアカウントの除外
- 登録状況の判定（登録済み／未登録）
- 新規登録用・更新用のExcelファイルを自動生成

使用技術：Python、DuckDB、pandas、openpyxl

※実行には別途ダウンロードデータが必要です。
